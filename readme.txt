1. Extract the projecFiles.zip file
2. Install Dependencies using python setup.py install
3. Move to the termProject directory
4. python sub_obj sub_obj_classification.py

This will run program outputing the precision and recall for each of the three classifiers as well
as a graph showing how the f_scores change for each classifier as the amount of features used
increases. Also a measures_recored.csv file will be created storing the precision and recall
measures.
